# HubSpot Deals Extraction API

## POST /api/v1/scan/start
Starts extraction.

## GET /api/v1/scan/status/{job_id}
Returns job status.

## GET /api/v1/scan/result/{job_id}
Returns extracted deals.

## POST /api/v1/scan/cancel/{job_id}
Cancels job.

## DELETE /api/v1/scan/remove/{job_id}
Deletes job.

## GET /api/v1/health
Health check.
